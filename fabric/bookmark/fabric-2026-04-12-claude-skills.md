# Claude Skills.

## How to set up Claude the right way (so you actually stop prompting).

AI has different levels:
- **Level 1**: Free ChatGPT
- **Level 2**: Paid ChatGPT + Thinking
- **Level 3**: Paid Claude chat + Opus + Thinking
- **Level 4**: Premium Claude + Cowork + Opus
- **Level 5**: Claude Teams with Projects

Today: **Skills**.

## 1 — What's different with Skills?

Skills fire _automatically_. Claude recognizes the task from what you type and activates the right Skill on its own. You don't invoke a Skill. _It invokes itself._

Context files need "read my file first" every time. Projects need you to open the right Project. But Skills fire automatically — no invocation needed.

## 2 — How to build your first Skill

### Option 1: Claude's built-in Skill Creator

Open Cowork → Use the skill-creator → Answer interview questions → Generate SKILL.md → Evaluate → Save → Install via Settings → Capabilities → Skills → Upload

### Option 2: makemyskill.com

Go to https://www.makemyskill.com, describe the skill, and download the generated SKILL.md file to upload to Claude.

## 3 — 7 Favorite Skills Hacks

1. **The debugging trick**: Ask Claude _"When would you use the linkedin-post skill?"_ — it quotes the description back, showing what's missing
2. **Negative triggers matter more than positive ones**: _"Do NOT use for..."_ prevents unwanted activation
3. **Skills stack with your voice file**: Voice file = who you are. Skill = how to do the job. They fire together.
4. **Build Skills from past conversations**: Convert old prompts into Skills via the Cowork chat menu
5. **Skills save tokens**: Claude only reads the 3-line header of each Skill at first. Full instructions only load on match.
6. **The "laziness" workaround**: Add _"Take your time. Quality over speed. Don't skip steps."_ to your prompt
7. **Skills are portable**: Anthropic published Skills as an open standard — the same SKILL.md works across platforms

## Where Skills Fall Short

- **The description is everything**: Bad description = Skill never fires
- **Skills can hijack conversations**: Fix with negative boundaries
- **Still needs editing**: Produces 80% there consistently, not perfection
- **Usage still burns fast**: Cowork still eats your plan at $100/month for heavy use

## Your First 30 Minutes with Claude Skills

- **0–5 min**: Open Cowork, find skill-creator, select Opus 4.6 + Extended thinking
- **5–15 min**: Answer the interview questions — be specific about your actual process
- **15–20 min**: Install and test
- **20–25 min**: Iterate — test 5 different phrasings
- **25–30 min**: Browse Claude's built-in plugins and install more skills
